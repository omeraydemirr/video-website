# About

This sample video website, built on PHP, is used for to manage basic video panel. It allows you to add,update or delete video.

## Tools and Technologies

* PHP
* Php-Admin
* Mysql

## Installation

```bash
sudo apt install php
```

```bash
cd video-website
```

```bash
php -S localhost:8000 -t src/
```
